Este projeto simula características de um blog de filmes onde os usuários podem ver a sinopse dos filmes e adicioná-los à sua lista pessoal de favoritos

#### Linguagens e Ferramentas
- PHP
- MySQL
- HTML, CSS, Bootstrap
- JavaScript
